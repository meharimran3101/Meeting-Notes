# Meeting to Action Notes

A modern, animated React web app that turns meeting transcripts into summaries, action items, key decisions, and risk highlights.

## Stack

- React 19
- Vite 8
- Custom CSS animations and glassmorphism UI

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## What the app includes

- Animated hero section with product-style UI
- Live transcript editor with sample meeting presets
- Simulated action extraction workflow
- Action board for owners, deadlines, decisions, and risks
- Responsive layout for desktop and mobile
